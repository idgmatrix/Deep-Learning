

# PyTorch 를 windows 10 에 설치하는 방법

* anaconda (python 3.6) 설치

* anaconda prompt 관리자 모드로 실행

* pytorch windows 10 용 설치

    * conda install -c peterjc123 pytorch cuda80 (GPU 지원)
